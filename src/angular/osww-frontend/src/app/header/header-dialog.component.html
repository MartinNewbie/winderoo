Reset takes 20 seconds

This screen will not close during the reset process

  1. Reset is complete when the second LED stops fast blinking, then turns solid.
  2. Connect your mobile device to the new WiFi network called, "Winderoo Setup"
  3. Add Winderoo to an existing WiFi network. If the LED starts to slow blink, Winderoo has successfully joined that network.
  4. Wait for your device to reconnect to your existing WiFi network.
